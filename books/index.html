ЕРЛАНГ КНИГИ

VOL.2

Друге видання КНИГИ N2O (англ. N2O BOOK Vol. 2 Green Book) визначає математичну специфікацію N2O ТЕХ на програмне забезпечення усіх рівнів прикладної моделі для підприємств та є вичерптним стартовим посібником для курсу навчання розробки технологічних програм для платформи Erlang.

VOL.3

Друге видання КНИГИ ERP (англ. ERP BOOK VOL.3 Blue Book) визначає формальну бізнес специфікацію та її імплементацію для сучасних оптимізованих підприємств. Системи ERP на її базі також уже не один рік використовується у банківській сфері, процесінгу транзакцій, розподілених системах повідомлень, в IoT секторі.

VOL.4

Друге видання КНИГИ MAD (англ. MAD BOOK VOL.4 Orange Book) розповідає про інструментальні засоби для розорбників, компіляція та збірка пакетів, різний автоматичний препроцесінг.

ЗАКРИТА КНИГА

Закрита книга це збірка попередніх версії N2O, ERP/BPE та MAD книг, які підходять для версій N2O від 0.11 до 4.4.

N2O: No Bullshit Sane Framework for Wild Web

BPE: Business Processing for Enterprise Done Right

MAD: Small and Fast Build Tools for Erlang Apps
