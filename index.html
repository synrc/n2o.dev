User Manuals

N2O.DEV is the Erlang/Elixir/N2O open-source foundation for ERP.UNO. Here you can find HTML/ROFF documentation and user manuals that cover family of N2O libraries. There are also: open source TeX/PDF books, release announcements and articles. All libraries are being built using Github Actions (mix, rebar3, mad), Erlang builds are checked against dialyzer. We do support last four major OTP versions: 21, 22, 23, 24 on Windows, Linux and Mac. Published at HEX.PM:
